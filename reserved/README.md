# Miyomi Reserved Redirect Page

This folder hosts the lightweight landing page we show while the main [Miyomi](https://miyomi.pages.dev) experience is deploying, rate-limited, or when we simply want to park a domain.

## Running locally

Any static server works:

```bash
npx serve .
# or
python -m http.server 4173
```

Open the printed localhost URL to preview.
